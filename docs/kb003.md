## Contexte

Déploiement du site statique (457 pages HTML) sur Cloudflare Pages, relié au repo GitHub `symbioticode/metiers-quebec.org-nouveau`.


## Problèmes rencontrés et solutions

### 1. `name` invalide dans wrangler.toml

**Erreur :**
```
Expected "name" to be of type string, alphanumeric and lowercase with dashes only
but got "metiers-quebec.org-nouveau".
```

**Cause :** Le `.` dans le nom n'est pas autorisé par Cloudflare.

**Solution :** Utiliser uniquement lettres minuscules, chiffres et tirets.
```toml
# ❌ Incorrect
name = "metiers-quebec.org-nouveau"

# ✅ Correct
name = 'metiers-quebec-org'
```

### 2. `[site]` bucket = Workers Sites, pas Pages

**Erreur :**
```
The entry-point file at "workers-site/index.js" was not found.
```

**Cause :** La config `[site] bucket = "./dist"` est pour Workers Sites, pas Pages.

**Solution :** Utiliser `[assets] directory = "dist"` pour Pages.
```toml
# ❌ Incorrect (Workers Sites)
[site]
bucket = "./dist"

# ✅ Correct (Pages)
[assets]
directory = "dist"
```

### 3. `wrangler deploy` vs `wrangler pages deploy`

**Erreur :**
```
It seems that you have run `wrangler deploy` on a Pages project,
`wrangler pages deploy` should be used instead.
```

**Cause :** La commande de build était `npx wrangler deploy`.

**Solution :** Supprimer la commande de build (laisser vide) — Cloudflare Pages sert les fichiers statiques directement depuis le répertoire `dist`.

### 4. Projet Pages inexistant

**Erreur :**
```
The Pages project "metiers-quebec-org" does not exist.
```

**Cause :** Le projet n'avait pas été créé sur Cloudflare.

**Solution :** Créer le projet dans le dashboard Cloudflare :
1. **Workers & Pages → Create application → Pages**
2. Connect to Git → sélectionner le repo
3. Build settings : command vide, output directory `dist`

### 5. `dist/` supprimé du repo

**Erreur :** Build vide — aucun fichier trouvé.

**Cause :** `dist/` avait été ajouté au `.gitignore` puis supprimé avec `git rm -r --cached dist/`.

**Solution :** Remettre `dist/` dans le repo (pas de `.gitignore` pour `dist/`).

`.gitignore` final :
```
public/
```
